Regra CSS

É o que define os elementos do CSS ficarem estilizados na página

O Seletor indica qual elemento HTML será estilizado pela regra aplicada dentro do <style>.

Exemplo: p, .container, #id1


A sintaxe básica de uma regra CSS é:

p{background-color: red;} = seletor{propriedade: valor;}

  1. Seletor: Indica a quais regras o CSS será aplicada
  2. Propriedade: é a caracteristica que voce deseja mudar
  3. Valor: Define como a propriedade deve se comportar
