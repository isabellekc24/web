O que é HTTP?

HTTP significa HyperText Transfer Protocol, ou em português, Protocolo de Transferência de Hipertexto. Ele é como uma língua comum que os computadores usam para se comunicarem na internet.

Como funciona?

Quando você entra em um site, por exemplo o www.google.com o que acontece é:

  • Seu navegador envia um pedido/requisiçãopara o servidor do Google.
    E este pedido segue as regras do protocolo HTTP.
  • O servidor do google recebe esta requisição, procura a página que você pediu e envia de volta a resposta.
  • Essa resposta chega ao seu navegador, que mostra o site na tela.
  • O navegador interpreta o conteúdo e exibe a página.
  • O HTTP serve para enviar e receber informações como: textos, imagens, sons, formulários, dados de uma página web
Observação: O HTTPS é uma versão segura do HTTP, isso significa que os dados enviados entre você e o site são criptografados.

O uso do HTTP é um ciclo:
Usuário → Navegador → Requisição HTTP → Servidor Web → Resposta HTTP → Página exibida

Requisição

Uma requisição HTTP é o que o navegador envia para o servidor.

			<GET /index.html HTTP/1.1
			<Host: www.exemplo.com
			<User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64)
			<Accept: text/html,application/xhtml+xml
			<Accept-Language: pt-BR
			<Connection: keep-alive
