O que é JavaScript?

JavaScript foi criado em 1995 por Brendan Eich para a Netscape, com o objetivo de adicionar interatividade dinâmica às páginas web estáticas, permitindo manipulação em tempo real de elementos e respostas a ações do usuário, diferenciando-se do HTML puro.

Ele pode ser executado tanto no navegador, onde roda no lado cliente para interagir com o DOM e a interface do usuário, quanto no servidor, via ambientes como Node.js, para lidar com lógica backend, acesso a bancos de dados e processamento de dados.

No navegador, o JavaScript opera em um "sandbox" isolado por razões de segurança, impedindo acesso direto ao sistema de arquivos ou hardware do dispositivo para proteger contra vulnerabilidades.

Objetos globais essenciais incluem document para manipular a estrutura da página, console para depuração e logging, e alert para exibir mensagens pop-up ao usuário.
