O que é HTML?

Sobre HTML:

HTML é uma linguagem de marcação para criar e estruturar páginas da WEB.

Ela define a estrutura e o conteúdo de um site, como parágrafos, títulos, imagens e links atráves de um conjunto de tags

Estrutura Básica HTML

  • <!DOCTYPE> - Indica que é um documento HTML5
  • <html> - É o elemento principal da página
  • <head> - Contém informações sobre a página como links, imagens, etc...
  • <body> - Contém o conteúdo visível da página
  • <title> - É o nome da página web que está sendo criada
