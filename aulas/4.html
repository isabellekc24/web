Formulários

O HTML possui a tag que possibilita definir formulários para a submissão de dados informados pelo usuário por meio de campos input, dentro do formulario. A tag <form> possui além dos atributos globais e também atributos action="url", utilizado para definir o destino dos dados submetidos. Exemplo: 

            <form action="url">
                <input type="text" name="variavelA        
            </form>"

Table

A tag table é utilizada para definição da tabela no HTML, internamente em uma tabela podemos criar linhas "rows" utilizando a tag <tr> e celulas dentro das linhas utilizando a tag <td> Exemplo: 

            <table>
                <tr>
                    <td: Celula 1x1 <td> Celula 1x2 </td>
                <tr>
                <tr>
                    <td: Celula 2x1 <td> Celula 2x2 </td>
                <tr>
             <table>
O código anterior define a tabela abaixo:
Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2
