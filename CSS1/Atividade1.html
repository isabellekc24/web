Uma regra CSS é a forma como você diz ao navegador como estilizar elementos HTML. Cada regra CSS indica quais elementos devem receber determinado estilo, qual propriedade desses elementos será alterada e qual valor será aplicado a essa propriedade.

CSS é composta por três partes principais:
- Seletor: indica quais elementos HTML a regra vai afetar.
- Propriedade: é o aspecto visual que você quer modificar (cor, tamanho, margem etc.).
- Valor: define como a propriedade será aplicada.
