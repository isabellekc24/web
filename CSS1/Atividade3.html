Atividade 3 – Como podemos incluir CSS em documentos HTML?


1- CSS Inline

O estilo é aplicado diretamente no elemento HTML usando o atributo style. Exemplo

Exemplo CSS Inline aplicando CSS dentro do elemento.


2- CSS Interno

O estilo é incluído dentro do próprio arquivo HTML, utilizando a tag <style> dentro do <head>


3- CSS Externo

O estilo é colocado em um arquivo separado com extensão .css e vinculado ao HTML usando <link> no <head>
